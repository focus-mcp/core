# Architecture Decision Records (ADR)

Ce dossier contient les ADR de FocusMCP au format [MADR](https://adr.github.io/madr/).

## Index

- [0001 — Documenter les décisions architecturales avec des ADR](./0001-record-architecture-decisions.md)
- [0002 — Développement en TDD strict](./0002-tdd-strict.md)

## Créer un nouvel ADR

1. Copier un ADR existant
2. Incrémenter le numéro
3. Remplir les sections (Contexte, Décision, Conséquences)
4. Ajouter à l'index ci-dessus
5. Soumettre via MR
